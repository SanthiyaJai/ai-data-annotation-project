# AI Data Annotation & Text Classification Project

## Overview
This project demonstrates hands-on experience in **data annotation, text classification, data cleaning, and quality evaluation tasks**.  
It simulates real-world AI training workflows used in machine learning dataset preparation.

---

## Key Tasks Performed

### 1. Text Classification
Classified text into structured categories based on meaning:
- Request
- Complaint
- Information

### 2. Data Cleaning
Performed data preprocessing tasks:
- Removed inconsistencies
- Standardized text formatting
- Corrected spelling errors
- Removed duplicate or noisy entries

### 3. Data Labeling
Applied rule-based labeling to ensure consistent dataset structure:
- Intent classification
- Category tagging
- Structured labeling format

### 4. Output Comparison
Compared multiple text outputs and selected the most accurate and clearly written version based on:
- Clarity
- Correctness
- Completeness
